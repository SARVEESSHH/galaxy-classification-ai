# 🌌 Galaxy Classification AI

A machine learning project that classifies galaxies into spiral and elliptical categories using astronomical image data.

## Objectives
- Build a convolutional neural network (CNN)
- Train on galaxy image datasets
- Visualize model performance
- Deploy results using GitHub Pages

## Tools
- Python
- TensorFlow
- Matplotlib
- Astronomy Image Data
